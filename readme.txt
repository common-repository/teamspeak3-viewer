== Description ==

To show the users and channels of a teamspeak3 server in a page
Shortcode:

[ts3view ip=ServerIp:QueryPort port=9987 logo=true|false gt=true|false]

== Installation == 

1. Unzip in /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
